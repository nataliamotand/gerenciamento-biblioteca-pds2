# gerenciamento-biblioteca-pds2

USER STORIES

1. Para o usuário, o sistema deve ter os seguintes recursos:
- Identificação de usuário, como nome de usuário e senha, para diferenciar e autenticar cada usuário.
- Lista de livros disponíveis para aluguel, com informações sobre título, autor, gênero e idioma.
- Sistema de aluguel de livros, permitindo aos usuários alugar um livro por um período de tempo definido.
- Sistema de devolução de livros, permitindo aos usuários devolver o livro após o uso.
- Histórico de aluguéis para que os usuários possam ver quais livros eles alugaram e quando eles foram alugados.
- Sistema de disponibilidade de livros, permitindo aos usuários verificar quais livros estão atualmente disponíveis para aluguel e quais estão alugados.
- Sistema de filtro para que os usuários possam pesquisar livros por gênero, autor e idioma.

2. Para atender às necessidades do administrador, o sistema deve ter os seguintes recursos:
- Sistema de gerenciamento de livros, permitindo ao administrador adicionar, editar e excluir livros disponíveis para aluguel.
- Sistema de gerenciamento de aluguéis, permitindo ao administrador ver quais livros estão atualmente alugados e quais usuários os alugaram.
- Sistema de gerenciamento de usuários, permitindo ao administrador adicionar, editar e excluir usuários, bem como visualizar seus históricos de aluguéis.
